## TEN PROJEKT TO PARODIA RZECZYWISTOŚCI
Każda osoba, która widzi to repozytorium zanim rozpocznie krytykę dowolnego elementu naszej "twórczości" chciałbym żeby wiedziała, że wszystko co tu jest jest półżartem, pół serio. :D

## Jak to włączyć

włącz ten skrypt z basha

```bash
  pip install -r requirements.txt
  cd Backend
  uvicorn Listener:app
```

i otwórzcie plik index.html w przegladarce (jest w /Frontend/index.html)

# podstawowe akcje

Zaloguj sie: login: `krzysiek`, hasło: `pizza`; lub stwórz nowego użytkownika

W miastach są firmy! Możesz kliknąć na miast i kupić akcje firmy w nim się znajdującej!

Co minutę są nowe newsy, które mają wpływ na rynek. Możesz je przeczytać na dole strony.

Za pewną cenę każdy gracz może podejrzeć status innego gracza, ale może mu się nie udać.
Można też najechać innego gracza, ale ponosi się ryzyko odwetu, które jest równie niebezpieczne co bycie najechanym.

# Przyszłość projektu

Wizja projektu jest szeroka. Gracze mogliby budować własne firmy, tworzyć skandale, podpisywać umowy pomiędzy swoimi firmami.

Design gry pozwala na eventy live-action takie jak np. zbiory plonów, wybory parlamentarne.

Szczególnie ważne są klęski żywiołowe, bo pozwalają sztucznie globalnie przychamować rozwój graczy oraz są alegorią do zmiany klimatu.

Rozbudowanie instrumentów finansowych o funkcjonalność shortowania, emitacji obligacji itp również ma niewyobrażalny potencjał.
